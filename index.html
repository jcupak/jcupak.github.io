Redesign - Inside & Out

I wrote my first web site with basic HTML, CSS, and JavaScript/jQuery way back in 1980 when Comcast allowed users to post their web pages. It had a lot of "flash-n-dash," as was typical for the web pages of the time. I used repeating background strips, animated icons, and special effects.

After Comcast dropped support for user web pages, I saved and archived my web site as I didn't want to pay for a web site. I recently discovered that I could host my web site on GitHub Pages, and spent some time updating the contents and styles of my web pages before rehosting them here.

This web site is a work in progress, and pages may change, some sections may not be complete or links to pages may not be implemented.

2023-04-07 Updated to Bootstrap 5.3.0-alpha and reorganized major sections.
